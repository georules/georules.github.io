Projects

The Code Lab

The Code Lab is exploring web front-end programming interfaces. It aims to close the distance between writing code, trying it, seeing results, and also finding code examples that solve small problems. It currently has two live demos:

  • Code Lab 0.1 A simple 3 text box python editor and executor. This version implements both the front end sending input and code to recieve output and/or errors, and a prototypal code execution api service.
  • Code Lab Web 0.1 A collaborative live HTML/CSS editor. This has explored the use of Firebase and ShareJS for OT data synchronization.

Tributary

enjalot's Tributary is a live javascript, coffeescript, and processing.js editor aimed at primarily utilizing d3.js to create data visualizations, geometrical drawings, and animations. It uses github gists to save and fork code from its users.

  • I have contributed to this project by adding in an in-browser screenshot capability, to add life to user pages and the recent inlets page.
  • I have also contributed to Inlet, tributary's codemirror plugin to utilize the thistle color picker.
  • I am currently working on implementing a loop depth counter to prevent infinite loops from crashing the application from being automatically evaluated.

yaplib

yet another php library is a simple set of tools to quickly mock up a restful api or power the backend of a web application. Started by ttricarico, yaplib started from the epiphany php framework with the goal of further simplifying the interface and adding more common tools for web development.

QuickClock Advanced Overclock

QCA by JPApps is an overclocking utility, and battery optimizer. QCA automatically determines the ideal scaling voltages and overclocked speeds for your device's processor as well as the maximum speed and implements safegaurds to reduce the possibility for any kind of damage.

QCA targeted android devices in the Droid 1 through X generations. It has not been updated to support any current dual-core or newer devices.

I created the framework for the user interface and stress tester. Paul Anderson integrated the overclocking module and designed the artwork.
